Analysis on Personal Investments for 2024

Python script comparing how my stocks performed, 2024-2025.

This python code uses yahoo finance to calculate and plot my total investments for 2024. The script also overlays how my investments would have performed if I invested all my money into each respective stock. Using this graph allows me to visualize which stocks are performing well for me and which ones I should sell.

This additional code reuses the dataframe generated to calculate the net gain or loss for each of my investments in 2024.
