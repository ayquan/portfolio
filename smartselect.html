Magic Scissors Selection Tool

CS2110 - Cornell University, Spring 2024.

This app allows the user to use a "magic scissors" tool that recognizes and outlines the objects being selected. The user can then cut the selected object out from the image and save it as a seperate pdf.

This is an example of how the tool can select and cut out an image.
